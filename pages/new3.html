Личный проект: сайт о городском транспорте

Идея проекта родилась из моего давнего увлечения автобусами и городским транспортом. Цель — создать простой, но удобный сайт, где можно:

  • Посмотреть актуальные маршруты автобусов
  • Узнать, какие модели используются на линиях
  • Оставить отзыв о комфорте перевозки

На данный момент разрабатывается макет в Figma и продумывается структура базы данных. Планируется использовать HTML, CSS и JavaScript для фронтенда, а в будущем — подключить простой backend (например, на Node.js или PHP) для хранения данных.

Этот проект — отличная возможность применить знания на практике и совместить учёбу с личными интересами.
